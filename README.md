##  Problemas Encontrados e Soluções
Problema 1: Concatenação no Flowgorithm
Problema: Inicialmente tentei usar + para concatenar texto, mas o Flowgorithm usa &.
Solução: Pesquisei na documentação e corrigi para "Olá, " & nome & "!".

Problema 2: Ordem dos parênteses no cálculo da média
Problema: Sem os parênteses, a média calculava errado (ex: 8 + 6 / 2 = 11 ao invés de 7).
Solução: Ajustei a fórmula para (nota1 + nota2) / 2, garantindo a precedência correta.

Problema 3: Esquecimento de declarar variáveis
Problema: Em alguns exercícios, tentava usar variáveis sem declarar.
Solução: Passei a declarar todas as variáveis no início do fluxograma.

Problema 4: Erro no primeiro push para o GitHub
Problema: Branch local era master e remota esperava main.
Solução: Usei git branch -M main para renomear e git push -u origin main.

Problema 5: Tipos de dados incorretos
Problema: Usei Inteiro para notas que poderiam ser decimais.
Solução: Mudei para tipo Real em todos os exercícios com números.
